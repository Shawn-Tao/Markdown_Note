# 泊松过程

## 计数过程定义

如果 $ N(t) $ 表示到时刻 $t$ 位置已经发生的时间 $A$ 的总数，且 $N(t)$ 满足条件

1. x
2. c
3. c
4. c
则称为计数过程

### 独立增量计数过程

对于
$$
t_1 < t_2 < ...< t_n
$$
有
$$
N(t_2) - N(t_1), N(t_3) - N(t_2),...N(t_n) - N(t_{n-1})  
$$
独立

### 平稳增量计数过程

事件发生只与间隔有关，与初始时刻无关

2. $X(t)$是独立增量过程
