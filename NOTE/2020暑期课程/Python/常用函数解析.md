## <font color = #996699>print</font>

print() 
中间加逗号会输出空格如，print("hello",i) => hello i

## <font color = #996699>range</font>

range(n) 生成 0 到 n-1 的整数序列 共 n个
range(m,n) 生成 M 到 n-1 的整数序列 共 n-m个

## <font color = #996699>pow</font> 

python中整数类型没有限制
pow(2,100) 也是可以算出来的
